💡 Projects

🔐 Project-Injection
A hands-on web security testing platform focused on learning and demonstrating vulnerabilities like XSS, HTML Injection, and SQL Injection. Includes an interface to try, test, and add payloads, useful for both beginners and intermediate learners in web application security.

🧭 RRU Navigator
A campus navigation and facility locator tool designed specifically for Rashtriya Raksha University. It helps students and visitors easily find academic blocks, hostels, cafeterias, and emergency points using a simple UI and integrated map support. Enhances campus experience for newcomers and event participants.

🏥 SiRa
A secure health record management system developed with privacy and accessibility in mind. SiRa stores patient records centrally, with features like emergency access, encryption, user verification, and admin oversight. The system supports both command-line (C++) and Flutter-based UI development.
