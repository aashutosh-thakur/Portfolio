💼 Hacktify - Cyber Security Intern

Worked remotely as a Cyber Security Intern at Hacktify, where I conducted in-depth testing and reporting of vulnerabilities across various labs and simulated environments.

  • Cross-Site Scripting (XSS): Exploited reflected and stored XSS vulnerabilities for alert pop-ups and data theft demos.

  • Insecure Direct Object References (IDOR): Accessed unauthorized resources by manipulating object references in URL parameters.
  • CORS Misconfigurations: Detected weak cross-origin policies that allowed sensitive data exposure and origin spoofing.
